ABOUT BF

BF is a tool designed for Exploratory Blockchain Analysis of the bitcoin ledger.

BECAUSE BITCOIN IS DISRUPTIVE...

Bitcoin is programmable money. Bitcoin is pseudonymous. Bitcoin is a shared public ledger.

BECAUSE BITCOIN IS DIFFERENT...

How to measure adoption within a pseudonymous system ? Can we really compare bitcoin with others payment systems by counting the number of transactions when a single bitcoin transaction can pay 1000 persons ? BF has no definitive answer to these questions but we expect it to be a first step on our way to a better understanding of bitcoin. BF provides visual and interactive tools helping users to explore the bitcoin ledger and figure out the dynamics taking place on the blockchain.

BECAUSE WE BETTER UNDERSTAND WHAT WE CAN MEASURE...

More than once, Bitcoin was described as a perfect tool for 'shadowy' activities requiring anonymity. Truth is that public ledgers have created a set of new challenges for the financial privacy of their users. Therefore, BF aims to define a set of metrics allowing a better evaluation of users privacy. BF supports the Open Bitcoin Privacy Project and its mission to improve financial privacy within the Bitcoin ecosystem.

BECAUSE INFORMATION IS POWER...

We believe that information resulting from data analysis of public ledgers should be accessible to the broadest public.

BF IS YOUR TOOL

Tell us which additional information / metrics features you would like to see in BF. Help us to improve the quality of information provided by BF and make it a better tool. Share your discoveries with the bitcoin community. Enjoy your journey in the blockchain !
